Protótipo site Cativa Operadora de Turismo

Informações básicas

  • Site de empresa operadora de pacotes turísticos
  • Desenvolvido em 2015/1
  • Projeto: http://rhuanbarros.github.io/cativaoperadora/
  • Site original: http://www.cativaoperadora.com.br/

Descrição do projeto

Protótipo construído apenas para fins de estudo sobre o desenvolvimento de nova apresentação da empresa na internet.

O objetivo era criar um novo design para o site da empresa objetivando o lançamento de uma plataforma de comércio eletronico.

Foi utilizado técnicas de SEO (Search Engine Optimization = Otimização para motores de busca) para a melhor colocação dos elementos visuais de modo a gerar maior conversão em vendas de pacotes de viagens turísticas

Utilizamos técnicas de avaliação heurística para determinar a melhor maneira para navagar no site de modo que o usuário encontra-se as informações o mais rápido possível e tivesse uma navegação intuitiva.

Mobile-first: O site foi todo construído utilizando o Framework Bootstrap e planejado para rodar da melhor maneira possível em dispositivos móveis.

Detalhes do projeto

  • HTML5 + CSS3 + JavaScript
  • Bootstrap
  • WooCommerce
  • Técnicas de usabilidade
  • Design apelativo
  • SEO (Search Engine Optimization)
